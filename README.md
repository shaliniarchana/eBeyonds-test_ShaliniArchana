# Associate Web Developer-Client Support – Technical Test

Developer - Shalini Archana Vidanaarachchi 

## Project Overview
This project is a simple responsive landing page developed as part of the **Associate Web Developer Technical Test**.  
The objective of this test is to demonstrate practical knowledge of modern web development technologies while following given instructions and design guidelines.



## Technologies Used
- **HTML5** – Semantic structure
- **CSS3** – Styling and layout
- **SASS (SCSS)** – Modular and maintainable styles
- **JavaScript (ES6)** – Interactivity (hamburger menu, language switcher)
- **Bootstrap 4** – Responsive grid layout
- **Swiper.js** – Slider component
- **Node.js** *(optional)* – Project initialization

## Features Implemented
- Responsive layout using Bootstrap 4 Grid
- News & Testomonials section slider using Swiper.js
- Hamburger menu for mobile navigation
- Language switcher using ES6 JavaScript
- Clean and organized SASS structure


## How to Run the Project
1. Open the project folder in Visual Studio Code
2. Ensure Live Server and Live Sass Compiler extensions are installed
3. Compile SASS using Live Sass Compiler
4. Open `index.html` using Live Server
